   📁 Dataset Information

    Dataset Name:   GTSRB 
        Number of Classes:   43 traffic sign categories

---

    Dataset Size

    Total Images:   8,600

    🔹 Split Distribution:

    Training Set:   6,020 images (70%)
    Validation Set:   1,290 images (15%)
    Test Set:   1,290 images (15%)

---

    Class Distribution

  All classes are   perfectly balanced  
  Each class contains:

      200 images total  
      140 images in training  
      30 images in validation  
      30 images in testing  

 This ensures:

  No class bias
  Fair model training
  Reliable evaluation

---

    Data Preprocessing Steps

The dataset was carefully prepared through the following steps:

  ✔️ Removal of corrupted images
  ✔️ Removal of duplicate images
  ✔️ Verification of readable images
  ✔️ Removal of empty classes
  ✔️ Creation of a balanced subset (200 images per class)
  ✔️ Dataset split into train/validation/test (70/15/15)
  ✔️ Image resizing to   64 × 64 pixels  
  ✔️ Label formatting using folder structure
  ✔️ Metadata file (`dataset_metadata.csv`) created
